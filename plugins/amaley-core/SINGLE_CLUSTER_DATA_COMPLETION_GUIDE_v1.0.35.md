# Single Cluster Data Completion Guide — v1.0.35

For complete public single-cluster pages, fill data in these places:

## Cluster
Amaley Core → Clusters:
- Short Introduction
- Full Story
- Main Products
- Villages
- Featured Image / Gallery Image URLs
- Contact Person / Phone if needed

## SHG Groups
Amaley Core → SHG Groups:
- Village
- District
- No. of Members
- Product Categories
- Short Story
- Full Story
- Verification Status
- Gallery Image URLs
- Contact Person / Phone if public display is intended

## Members / Producers
Amaley Core → Members / Producers:
- Role
- Skills
- Products Handled
- Short Bio
- Story
- Village
- Phone if public display is intended
- Photo URL or Featured Image

## Product Origin Mapping
WooCommerce product → Amaley Origin:
- Primary Cluster
- Linked SHGs
- Linked Producers
- Source Village / Region
- Origin Note

v1.0.35 also extends CSV import/export columns so these detail fields can be filled in bulk.
