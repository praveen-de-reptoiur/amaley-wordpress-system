# Cluster Archive Section Widgets Guide v1.0.27

Build the Clusters archive page in Elementor using separate Amaley Core widgets in this order:

1. Amaley Cluster Archive Hero
2. Amaley Cluster Archive Trust Strip
3. Amaley Cluster Archive Intro / Why Section
4. Amaley Cluster Archive Grid
5. Amaley Cluster Archive CTA Band

Do not use the old all-in-one archive widget as the final workflow. It remains only for legacy/testing.

## Editing

Each widget has section-wise Elementor controls:

- Content
- Breadcrumbs / links
- Buttons
- Data query
- Card elements
- Layout
- Style

Data still comes from:

- Amaley Core → Clusters
- Amaley Core → SHG Groups
- Amaley Core → Members / Producers
- Amaley Core → Product Origin Mapping
