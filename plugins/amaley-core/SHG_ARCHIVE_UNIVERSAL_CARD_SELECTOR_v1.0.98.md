# SHG Archive Universal Card Selector v1.0.98

## Scope
Adds lightweight universal card selector to:
- Amaley SHG Archive Grid

## New control
5. Card Template / Content Controls:
- Current / Existing Card
- OG SHG Card 1

## What OG SHG Card 1 follows
Universal card flow:
- image / initials placeholder
- label
- title
- description
- meta/stat boxes
- tags/chips
- full-width rounded button

- Show Image / Placeholder Area

- Show Verified Badge
- Show Card Description
- Show Cluster Detail
- Show Location Detail
- Show Members Detail
- Show Status Detail Box
- Show Contact Detail
- Show Product/Work Tags
- Show Card Button

- Card / Box Style
- Image / Placeholder Style
- Card Text Style
- Details / Meta Boxes Style
- Tags / Badges Style
- Button Style
- Alignment controls

- No OG full controls
- No new transform controls
- No archive JS/AJAX
- No pagination change
- Single Cluster, Single SHG, Single Member, Cluster Archive, Discovery Engine, WooCommerce, header/footer untouched
