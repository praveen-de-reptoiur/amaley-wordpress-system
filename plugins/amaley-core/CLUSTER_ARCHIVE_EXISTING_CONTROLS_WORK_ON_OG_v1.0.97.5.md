# Cluster Archive Existing Controls Work on OG Card v1.0.97.5

## Issue
The controls visible in Elementor sections 8–13 were targeting only the old/current Cluster Archive card classes. In OG Cluster Card 1 mode, those controls looked present but did not affect the OG card.

## Fix
No new heavy controls were added. Existing controls were mapped to both the current card classes and the lightweight OG card classes.

## Existing controls now mapped to OG card
- 8. Card Outer Box
- 9. Image / Fallback Area
- 10. Card Text Elements
- 11. Product Tags
- 12. SHG / Producer Stat Boxes
- 13. Card Button

## Still lightweight
- No OG full controls
- No transform controls
- No new archive JS/AJAX
- No pagination changes
- No changes to Single Cluster, Single SHG, Single Member, Discovery Engine, WooCommerce, header/footer
