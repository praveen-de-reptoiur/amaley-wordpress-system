# Rich Text Cluster Story Field — v1.0.45

The Cluster Story section on the single cluster template uses the actual cluster data field:

`Amaley Core → Clusters → Story → Full Story`

This field is now a rich editor.

You can add:
- paragraphs
- bold text
- italic text
- bullet lists
- numbered lists
- blockquotes
- links

Do not use Elementor widget custom story override. That override was removed to avoid conflict.

Workflow:
1. Open the Cluster in WordPress admin.
2. Go to Story → Full Story.
3. Add formatted content.
4. Save/Update the cluster.
5. Refresh the single cluster page.

The Elementor widget only controls layout and styling.
The actual story content comes from cluster data.
