# Amaley Core v1.0.52 — SHG Archive Font, Card and Control Polish

## Purpose

This version tightens the SHG Archive after frontend review.

## What changed

- Removed duplicated Status detail row from cards when the verification badge is already visible.
- Added `Story Words Per Card` control for card description length.
- Kept `Max Tags Per Card` control for chip count.
- Tightened card proportions for 4-column archive layout.
- Reduced image/placeholder height and made placeholders less dominant.
- Strengthened hero button visibility against theme CSS.
- Improved dynamic hero stat fallback so old Local/Origin/Small/Care values are replaced by meaningful counts.
- Cleaned the second explanation section copy so its purpose is clearer.
- Set safer scoped typography defaults for SHG Archive sections.

## Preserved

- v1.0.49 SHG grid query fix preserved.
- v1.0.48 Elementor registration fallback preserved.
- v1.0.47 SHG Archive/Single widgets preserved.
- SHG Single widgets are not redesigned in this version.
- WooCommerce cart/checkout untouched.
- Header/footer untouched.
- Routes/permalinks untouched.
