# Amaley Manual Gallery Section - v1.0.76.5

Status: test plugin update.

This version adds a reusable Elementor widget named **Amaley Manual Gallery Section**.

Use this widget when a page/archives needs manually selected images inside Elementor, for example:
- Cluster Archive
- SHG Archive
- Member / Producer Archive
- Home page
- Our Story / Ecosystem pages
- Any Amaley landing section needing manual visual references

Important lock:
- Archive/page gallery = manual Elementor images.
- Single CPT gallery = images fetched from the individual Cluster / SHG / Member record fields.

No locked Cluster, SHG, Member, or Product card design was changed in this version.
