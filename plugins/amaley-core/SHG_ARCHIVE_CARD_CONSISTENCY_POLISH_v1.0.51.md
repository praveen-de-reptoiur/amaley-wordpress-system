# Amaley Core v1.0.51 — SHG Archive Card Consistency Polish

## Purpose

This version polishes the SHG Groups Archive after live Elementor review.

## What changed

- Keeps SHG Archive widgets working from v1.0.49/v1.0.50.
- Updates hero stats to show meaningful dynamic counts instead of generic words.
- Fixes primary hero button visibility on light background.
- Makes SHG archive cards visually closer to the SHG cards used inside Cluster Single sections.
- Adds compact detail grid on SHG archive cards: Cluster, Location, Members, Verification and Contact where available.
- Keeps verification badge visible as a proper small horizontal pill.
- Reduces card size and sets default desktop grid to 4 columns.
- Reduces placeholder image height and uses initials instead of a large “SHG” wordmark.
- Keeps product chips to a compact single-row feel.

## What did not change

- No SHG Single build change.
- No WooCommerce checkout/cart change.
- No header/footer change.
- No route/permalink change.
- No Discovery Engine change.
- No CPT/meta/data migration.

## Preserved fixes

- v1.0.48 Elementor widget registration fix.
- v1.0.49 SHG Archive grid query fix.
- v1.0.50 control foundation.
