# Cluster OG Card Visibility + Transform Controls v1.0.89

## Purpose
Complete the Cluster Single OG card control layer by adding:
- card element show/hide controls
- card transform/motion controls

## Scope
Only these accepted Cluster Single widgets:
- Amaley Cluster Related SHGs
- Amaley Cluster Related Producers

## New Content controls
Content → Card Elements / Show-Hide:
- Show Image / Placeholder
- Show Card Label
- Show Card Title
- Show Card Description
- Show Meta / Stat Boxes
- Show Tags / Chips
- Show Card Button

## New Style controls
Style → Card Transform / Motion:
- Translate Y
- Scale
- Rotate
- Hover Translate Y
- Hover Scale
- Hover Rotate
- Transition Duration

## Safety
- Default values keep the accepted OG card design unchanged.
- No Member Single changes.
- No product card migration.
- No Discovery Engine changes.
- No WooCommerce logic changes.
- No header/footer changes.
