# Member Archive Lightweight OG Selector v1.0.99.3

## Scope
Adds a lightweight OG Member Card 1 option to the Member / Producer Archive Grid after Atomic Editor was deactivated.

## Updated widget
- Amaley Member Archive Grid

## New control
2. Card Template + Elements + Button Content:
- Current / Existing Card
- OG Member Card 1

## Controls working in OG mode
- Show Image / Placeholder
- Show Placeholder When No Image
- Show Card Label
- Show Role Detail
- Show Village Detail
- Show SHG Detail
- Show Cluster Detail
- Show Skill Tags
- Show Product Tags
- Show Bio / Description
- Show Card Button
- Show Section Button

## Important safety decision
Existing style controls are intentionally not force-mapped to OG Member Card 1 in this version. This avoids the previous Elementor left-panel loading issue. OG Member Card 1 uses the shared universal fixed CSS.

## Not included
- No heavy OG full controls
- No transform controls
- No new archive JS/AJAX
- No pagination changes
- No style selector mapping to OG card

## Base
Built from v1.0.99.2 rollback/editor-safe build.
