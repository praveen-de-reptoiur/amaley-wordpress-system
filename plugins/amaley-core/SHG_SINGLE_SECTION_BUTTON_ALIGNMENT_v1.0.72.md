# SHG Single Section Button Alignment — v1.0.72

Purpose: fix the section-level buttons such as “View All Products”, “View All Producers”, and “View All Clusters” so they remain visible and controllable without changing the locked card designs.

## What changed

- Section-level action buttons now use a visible rust-filled default style instead of blending into the ivory page background.
- Added dedicated Elementor control: **Section Button Alignment**.
- Added Elementor controls for section button background, text color, hover background and hover text color.
- Cluster Single section action links also receive the same visibility safety.

## What was not changed

- Product card layout and pricing/meta alignment.
- SHG card design.
- Member/producer card design.
- Cluster card design.
- Header, footer, WooCommerce cart/checkout/order logic, or origin mapping logic.
