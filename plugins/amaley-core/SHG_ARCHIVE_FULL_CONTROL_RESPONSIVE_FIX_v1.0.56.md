# SHG Archive Full-Control Responsive Fix — v1.0.56

Status: Test build after user feedback.

## Why this version exists

v1.0.54 caused Elementor editor instability. v1.0.55 restored the stable baseline but still did not fully follow the Universal Full-Control Website Standard for SHG Archive Grid controls and phone responsiveness.

## Scope

- SHG Archive Grid widget only.
- Adds clearer Elementor control groups.
- Adds phone responsive CSS safeguards.
- Keeps v1.0.49 grid query fix.
- Keeps v1.0.53 gallery base.
- Does not touch SHG Single structure.
- Does not touch WooCommerce, header, footer, routes or Discovery Engine.

## Control groups added / cleaned

Content tab:

1. Heading Content
2. Show / Hide Elements
3. Data / Query / Source
4. Links / Routing
5. Card Content Controls
6. Layout / Responsive Columns

Style tab:

7. Section Background
8. Section Spacing
9. Heading / Label Style
10. Card / Box Style
11. Image / Placeholder Style
12. Card Text Style
13. Details / Meta Boxes Style
14. Tags / Badges Style
15. Button Style
16. Responsive Fine Tuning

## Important controls

- Columns Desktop / Tablet / Mobile
- Image / Placeholder Height
- Image Opacity
- Placeholder Circle Size
- Description Words Per Card
- Description Line Clamp
- Max Tags Per Card
- Details Gap / Padding / Colors
- Button padding / radius / colors / hover colors
- Section padding and grid gap

## Safety

No broad global CSS. All frontend CSS remains scoped to Amaley Core SHG archive classes.
