# Cluster Section Card Template Selector v1.0.87

## Purpose
This version adds a simple section-level card template selector only to the accepted Cluster Single related-card widgets.

## Widgets updated
- Amaley Cluster Related SHGs
- Amaley Cluster Related Producers

## New Elementor control
Content → Related Section Content → Card Template

Options:
- OG Card 1
- Current / Existing Card

## Default
OG Card 1

This preserves the currently accepted frontend because the Cluster Single SHG and Producer sections are already accepted as universal OG card designs.

## Scope safety
- Member Single is untouched.
- Product cards are untouched.
- Discovery Engine is untouched.
- WooCommerce is untouched.
- Header/footer are untouched.
- No preview widget added.
- No new frontend redesign.
- Existing accepted Cluster Single card visuals remain the same.

## How to use
Open the Cluster Single template in Elementor.
Select either:
- Amaley Cluster Related SHGs
- Amaley Cluster Related Producers

Then choose:
Card Template → OG Card 1
