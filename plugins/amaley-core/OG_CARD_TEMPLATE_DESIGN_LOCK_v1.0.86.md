# OG Card Template Design Lock v1.0.86

## Purpose
This version locks the existing accepted card designs as named OG templates without changing the card visuals.

## Locked templates
- Cluster → OG Cluster Card 1
- SHGs → OG SHG Card 1
- Member → OG Member Card 1
- Product → OG Product Card 1

## Important
No card design has been changed in this version.
The cards are locked as the approved baseline so Elementor section-level selectors can later use these templates safely.

## Card flow
All OG Card 1 templates follow the same structure:

1. Media / image / initials placeholder
2. Label / eyebrow
3. Title
4. Description / excerpt
5. Meta/stat boxes
6. Tags/chips
7. Primary button

## Future-proof slots
Each template definition keeps add-on slots for future needs, such as:
- badge
- icon
- verification
- extra meta
- secondary button

## Future template naming
If a new design is approved later, it should be added as:
- OG Cluster Card 2
- OG SHG Card 2
- OG Member Card 2
- OG Product Card 2

Do not unexpectedly alter OG Card 1 after it is accepted.

## Scope safety
- Existing frontend pages do not change automatically.
- No Member Single renderer connection.
- No preview widget.
- No product card migration.
- No Discovery Engine change.
- No WooCommerce logic change.
- No header/footer change.
- Existing accepted Cluster Single SHG/Producer card work remains intact.
