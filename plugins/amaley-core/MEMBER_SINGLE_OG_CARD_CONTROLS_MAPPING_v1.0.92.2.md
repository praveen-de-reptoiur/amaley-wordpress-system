# Member Single OG Card Controls Mapping v1.0.92.2

## Purpose
Maps Member Single Elementor style controls to the approved OG Card 1 classes, following the same direction already accepted on Cluster Single.

## Widgets updated
- Amaley Member Single Linked SHG
- Amaley Member Single Linked Cluster
- Amaley Member Single Products

## Controls now mapped to OG Card 1
Existing style controls now target central OG card classes:
- `.amaley-card`
- `.amaley-card__media`
- `.amaley-card__initials`
- `.amaley-card__label`
- `.amaley-card__title`
- `.amaley-card__excerpt`
- `.amaley-card__body`
- `.amaley-card__meta`
- `.amaley-card__meta-item`
- `.amaley-card__tags`
- `.amaley-card__button`

## New Style section
A new section was added:
`OG Card: Tags / Inner Gap / Transform`

It includes:
- Card inner gap
- Tags/chips gap
- Tags typography
- Tags text/background/border/padding/radius
- Translate Y
- Scale
- Rotate
- Hover Translate Y
- Hover Scale
- Hover Rotate
- Transition duration

## Safety
- Default values do not visually change the approved OG cards.
- Cluster Single v1.0.91 remains untouched.
- No pagination change in this patch.
- Discovery Engine is untouched.
- WooCommerce logic is untouched.
- Header/footer are untouched.
