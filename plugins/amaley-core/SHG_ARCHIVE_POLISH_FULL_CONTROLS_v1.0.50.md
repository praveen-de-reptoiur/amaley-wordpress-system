# Amaley Core v1.0.50 — SHG Archive Polish + Full Controls

## Purpose

This version polishes the SHG Archive page before continuing to SHG Single templates.

The archive widgets already rendered after v1.0.49, but the page needed better default copy, cleaner card behaviour and proper non-coder controls in Elementor.

## What changed

- Added detailed Elementor controls for SHG Archive Hero, Trust Strip, Intro, Grid and CTA.
- Added content controls for labels, headings, descriptions, buttons, stats and feature cards.
- Added show/hide controls for each important visible element.
- Added query controls for grid limit, cluster ID, featured only, website visibility, verification status, order by and order.
- Added card-level controls for image area, placeholder, verification badge, cluster, location, member count, story, product tags and button.
- Added layout/spacing controls for section padding, width, gaps, card padding, radius and image height.
- Added color controls for major section/card/text/button elements.
- Improved missing-image behaviour so cards do not look like large empty blocks by default.
- Fixed dark CTA title contrast.

## What did not change

- No WooCommerce cart or checkout change.
- No header/footer change.
- No Discovery Engine change.
- No route/permalink change.
- No SHG Single final polish yet.

## Testing focus

1. Activate v1.0.50.
2. Open the SHG Groups archive page in Elementor.
3. Confirm all SHG Archive widgets still render.
4. Confirm the grid shows SHG cards.
5. Confirm controls are available in Content and Style tabs.
6. Confirm cards look compact when SHG records do not have featured images.
