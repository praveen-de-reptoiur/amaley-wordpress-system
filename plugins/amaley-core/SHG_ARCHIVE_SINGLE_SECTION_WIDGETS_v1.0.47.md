## Workflow

Use one Elementor page/template with multiple Amaley Core section widgets. Do not use one hardcoded all-in-one final widget.

## Safety

- No WooCommerce cart/checkout override
- No header/footer override
- No permalink rewrite
- No Discovery Engine change
- No relation/meta migration
