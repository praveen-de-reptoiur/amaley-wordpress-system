# Amaley Compact Widgets

Professional compact Elementor widgets and shortcodes for Amaley visual sections.

## Version
0.4.5-test

## Real map repair in this build
- Replaced fake/dull map texture with real OpenStreetMap tile rendering.
- Added working plus/minus zoom controls.
- Added drag/pan interaction for mouse and touch.
- Added mouse-wheel zoom.
- Added reset button.
- Kept all code scoped to the Origin Map Path widget.
- Uses vanilla JavaScript only; no Leaflet/Google Maps library is bundled.

## New widget
- Elementor widget: Amaley Origin Map Path
- Shortcode: `[amaley_cw_origin_map]`

## Safety
This plugin does not touch Discovery Engine, Amaley Core, Amaley Templates, Site Shell, WooCommerce cart/checkout, header/footer or CPT data logic.

## Note
Real map tiles require internet access and access to the configured tile server. Default tile server is OpenStreetMap.
