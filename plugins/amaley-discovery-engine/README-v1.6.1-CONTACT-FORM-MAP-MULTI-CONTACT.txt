Amaley Discovery Engine v1.6.1
Contact widgets polish and Contact Form CTA

Changes:
- Contact Info Cards now support multiple contact lines per card.
  Add one phone/email/address per line in "Multiple Contact Lines".
  Phone, email and URL lines can auto-link.
- Contact Map Section now supports Address Auto Map.
  Add address/search query and it renders a Google Maps embed without API key.
  Existing iframe, image fallback and shortcode options remain.
- Contact Map phone/email controls support multi-line text.
- New widget: Amaley Contact Form CTA.
  Supports built-in demo/mailto form, Contact Form 7/WPForms/Fluent Forms shortcode, or safe HTML embed.
  Includes desktop/tablet/mobile visibility controls and full Elementor-native style controls.
- Mobile-first defaults added for cards, map and form CTA.

Safe scope:
- Legacy widgets are not restored.
- Product filter widget is not changed.
- WooCommerce/product query logic is not changed.
- Shop Hero, Shop Strip and Universal CTA are not changed except shared CSS file extension.
