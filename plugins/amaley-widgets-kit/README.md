# Amaley Widgets Kit

Elementor-only reusable widgets plugin for the Amaley WordPress System.

## Purpose

Amaley Widgets Kit will provide pre-built Elementor widgets for Amaley pages, product sections, story sections, CTAs, cards, origin blocks, trust sections, and reusable design layouts.

## Status

Planning / audit stage.

## Rules

- Widget-only plugin.
- No CPT creation.
- No WooCommerce replacement.
- No header/footer replacement.
- No discovery/filter engine replacement.
- Scoped CSS only.
- Elementor-native controls.
- Mobile-first.
- Non-coder manageable.
- Every widget must include 2–3 design presets.
- Every widget must include proper content, style, spacing, typography, layout, and responsive controls.

## Planned Phase 1 Widgets

1. Amaley Section Heading
2. Amaley Page Hero
3. Amaley CTA Band
4. Amaley Info Cards
5. Amaley Media + Text Split
6. Amaley Product Collection Cards
7. Amaley Product Card

## Preset Styles

1. Editorial Premium
2. Marketplace Clean
3. Community Story

## Development Rule

Do not build all widgets at once. Start with Phase 1 only, test on staging, document, then move to Phase 2.
