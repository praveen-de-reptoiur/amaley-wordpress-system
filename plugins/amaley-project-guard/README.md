## Safety

- Separate plugin.
- Not inside Amaley Core.
- No frontend output.
- No frontend CSS/JS.
- No auto-fix.
- No deletion.
- No plugin activation/deactivation.
- Manual scan only.
- Amaley Core is only a read-only scan target.
