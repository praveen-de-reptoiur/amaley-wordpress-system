# Amaley Project Guard v1.0.0 — Clean Rebuild Lock

This v1.0.0 is a fresh separate-plugin build.

## Hard lock

- Separate plugin folder: `amaley-project-guard`
- Separate top-level admin menu: `Amaley Project Guard`
- No submenu under Amaley Core
- No dependency on Amaley Core
- No inclusion of Amaley Core files
- No Amaley Core class/function call required for plugin boot
- Amaley Core is only scanned as a read-only target if installed
- No frontend output
- No frontend CSS/JS
- No auto-fix
- No deletion
- No plugin activation/deactivation
- Manual Quick Scan only
- Cached last report

## v1.0.0 scope

- Admin dashboard

- Amaley Core target file/version checks without dependency
- Elementor status and Atomic feature warning
