# Amaley H/F Studio V2

Version tracked here: 2.0.4 Hide/Show Controls.

Purpose: Elementor-style header/footer studio for Amaley with live-style widgets and assignment rules.

Current status:
- Header and footer rendering tested on staging.
- v2.0.4 adds non-coder hide/show controls for header/footer sections and elements.
- Style/layout controls are next phase.

Safety rules:
- Do not activate old Amaley H/F or old Amaley Site Shell together with this plugin.
- Theme hiding stays OFF unless tested.
- No output buffering.
- Frontend rendering uses wp_body_open and wp_footer only.
