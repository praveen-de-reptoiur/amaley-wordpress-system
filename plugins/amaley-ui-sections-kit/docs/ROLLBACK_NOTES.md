# Rollback Notes — Amaley UI Sections Kit v0.1.2

## Simple Rollback

1. Deactivate the plugin.
2. Clear cache.
3. Remove shortcodes from test page if needed.

## Database Rollback

No database rollback is required. This version creates no tables, CPTs, options, product data, origin data, SHG data or WooCommerce changes.

## Live Site Warning

Do not place these shortcodes into live homepage/product/shop pages until the components have been reviewed on staging or a private test page.
